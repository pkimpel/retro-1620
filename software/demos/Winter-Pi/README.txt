Index of retro-1620/software/demos/Winter-Pi:
Program to calculate digits of Pi.

Winter-Pi is a version of Dik Winter's 800-digits-of-Pi program for the
IBM-1620. On a 20K-digit system it can generate up to 640 digits of Pi.
See https://crypto.stanford.edu/pbc/notes/pi/code.html.

Winter-Pi.sps
    Source deck for the card-based SPS assembler.

Winter-Pi.lst
    Assembly and typewriter output for the program.

Winter-Pi-Object.card
    Bootable object card deck for the program.

    Full (non-condensed) bootable object card deck for the program.
