Index of retro-1620/software/diagnostics/DT0021:
IBM 1620 diagnostic routine for the 1311 disk drive.

This program will write test data on a 1311 disk pack in preparation for
running the DT0022 diagnostic program.
See http://bitsavers.org/pdf/ibm/1620/fe/diag/2161819_DT21_Apr65.pdf.
