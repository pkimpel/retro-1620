IBM 1620 diagnostic routine for the 1311 disk drive.

This program will perform a comprehensive test of the 1311 disk drive
and 1620 interface circuits.
See http://bitsavers.org/pdf/ibm/1620/fe/diag/2161822_DT22_May64.pdf.
